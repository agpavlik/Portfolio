# Portfolio

Portfolio website created with React and Tailwind.

## Features of App

- Dark mode
- Modal window
- Set of selection buttons
- 3d model with Three.js
- Animation with AOS
- Responsive web design

## Tech Stack

- React
- React Router
- Tailwind
- React-Icon
- Three.js
- AOS
